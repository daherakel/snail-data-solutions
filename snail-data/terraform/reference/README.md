# Reference snapshots

Solo lectura. Exportes/config actuales para apoyar IaC, no se aplican directamente:
- `apigw/exports`: Swagger exports de APIs
- `apigw/stages`: metadata de stages/deployments
- `lambda/configs`: configuraciones de Lambdas desde AWS CLI
- `iam/roles`: dump de roles
- `iam/policies/{attached,inline}`: listados/dumps de policies
- `glue/jobs`: config de Glue jobs
- `sfn/definitions`: definiciones ASL de state machines
